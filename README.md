Pantheon is a code similarity detection tool designed to compare student programming submissions and identify potential plagiarism. The application performs structural and syntactic analysis of source code to calculate similarity metrics between multiple files, reducing reliance on basic text-based comparison.

The system aims to support academic integrity by providing an efficient and extensible platform for automated code comparison. Pantheon emphasizes modular design, usability, and accurate similarity reporting, making it suitable for use in educational environments.
